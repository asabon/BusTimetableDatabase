# プロジェクト理解

## 概要
- 神奈中バスの時刻表データベース。
- Webスクレイピングでデータを取得し、GitHub Actionsで定期更新する。
- **データ形式**: JSON。詳細は README.md および docs/v3/README.md を参照。
- **想定利用者**: 神奈中バスの時刻表データを利用したいアプリ開発者。

## バージョニング原則 (VER)
- データベースフォーマット（スキーマ）変更時は新しいバージョンディレクトリを作成する。
- 利用アプリ側の互換性維持のため、アプリの移行が完了するまで旧バージョンを並行運用する。
- 移行完了後は速やかに旧バージョン（データ、スクリプト、ワークフロー）を削除する。
- 現在は v3 が最新かつ唯一の運用バージョンであり、v1, v2 は削除済み。

## 主要ドキュメント
- docs/v3/README.md: v3 データベースの内部仕様。
- README.md: プロジェクト全体の概要。
