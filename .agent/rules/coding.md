# コーディング規約 & ファイル構成

## Python 実行環境
- **Python バージョン**: 3.13.x
- **パッケージ管理**: `uv` を使用している。
- **仮想環境**: リポジトリルートの `.venv` ディレクトリを使用する。
- **有効化コマンド (PowerShell)**: `& .venv/Scripts/Activate.ps1`

## コーディング規約 (Lint & Format)
コードの品質と一貫性を保つため、以下の手順を**必須**とする。

1. **自動フォーマット**: コード修正後、必ず `ruff format .` を実行すること。
2. **自動修正とチェック**: `ruff check . --fix` を実行し、可能な限り自動修正を行うこと。
3. **最終確認**: 完了報告の前に必ず `ruff check .` を実行し、すべての指摘が解消されている（All checks passed!）ことを確認すること。指摘が残っている状態での完了報告は認められない。
