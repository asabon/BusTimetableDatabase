# /resume ワークフロー

このワークフローは、`docs/progress/task.md` から前回の作業状態を読み込み、現在のコンテキストに復元します。

## 実行手順

1. `/init` を実行し、基本ルールをロードする。
2. `docs/progress/task.md` を読み込む。
3. 前回の完了事項と「次回へのメモ」を把握する。
4. 現在の作業状態を要約し、ユーザーに「前回の続きから再開します」と報告する。
5. `task.md` の未完了タスク（`[ ]` または `[/]`）に基づき、次のアクションを開始する。

## 完了条件

- 作業状況が復元され、次のタスクが特定された。
- ユーザーに再開を報告した。
