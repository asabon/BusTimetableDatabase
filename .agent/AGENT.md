# AGENT 指示書 (Index)

このファイルは Antigravity エージェントの動作の起点となるインデックスである。
ターン中、必要に応じて各詳細ルールを参照すること。

## 1. 毎ターン必須の基本ルール
- **GEN-000**: エージェントは**毎ターンこのファイルを読み込み**、その後で、必ず「.agent/AGENT.md を読み込みました」とユーザーに報告すること。
- **GEN-001**: すべての返答は必ず日本語で行うこと。
- **GEN-002**: 指示の背景や詳細を把握するため、必要に応じて `.agent/rules/` 配下の詳細ルールを読み込むこと。
- **GEN-003**: 開発フローは **`github-dev-flow` スキル** を主として使用し、進捗管理はプロジェクト内の `docs/progress/task.md` を Source of Truth とする。作業開始時や進捗があった際は必ずこのファイルを確認・更新すること。
- **GEN-004 (報告)**: 実装完了時は、**Issue のすべてのタスク（チェックボックス）が完了し、`task.md` が削除されていることを確認した上で**、Pull Request または Issue で成果を報告すること。

## 2. 動作最適化ルール (Performance & Operations)
- **OPT-001 (読込)**: 大きなファイルはまず `view_file_outline` で構造を把握し、必要な範囲のみを読み込むことでコンテキストの消費を抑える。
- **OPT-002 (記憶)**: セッション間で保持すべき情報は、内部メモリや適切な永続化手段を活用し、効率的に再利用する。
