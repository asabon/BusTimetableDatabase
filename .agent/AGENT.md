# AGENT 指示書 (Index)

このファイルは Antigravity エージェントの動作の起点となるインデックスである。
エージェントは**毎ターンこのファイルを読み込み**、必ず「.agent/AGENT.md を読み込みました」とユーザーに報告すること。
そのターン中、必要に応じて各詳細ルールを参照すること。

## 1. 毎ターン必須の基本ルール
- **GEN-001**: すべての返答は必ず日本語で行うこと。
- **GEN-002**: 指示の背景や詳細を把握するため、必要に応じて `.agent/rules/` 配下の詳細ルールを読み込むこと。

## 2. 詳細ルール目次 (Table of Contents)
具体的な動作ガイドラインは以下の各ファイルに集約されています。実行中のタスクに関連するものを適宜参照してください。

- [**プロジェクト理解・構成**](rules/prj.md)
  - プロジェクト概要、v3 運用、バージョニング原則
- [**Git 管理**](rules/git.md)
  - コミットメッセージ、ブランチ規約、Git コマンド
- [**GitHub 連携**](rules/github.md)
  - Issue・PR 管理、GitHub CLI (gh) の扱い
- [**パフォーマンス最適化**](rules/perf.md)
  - モード切り替え、ファイル読み込み効率化、キャッシュ
- [**コーディング・ファイル規約**](rules/coding.md)
  - Python 規約 (ruff)、ディレクトリ構造、大量データの扱い
